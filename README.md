This is a simple python project for managing the history of Ipython (Interactive python)

IPython is an interactive version of python that runs inside the terminal. It is incredibly
usefull to quickly test small lines of code or run small calculations. BUT did you know, ipython
stores every single line of code you run and you still have to import scientific constants!!.

this project aims to stop storing every line and automatically import scientific constants and modules
.... or any variables and modules the user wants.
